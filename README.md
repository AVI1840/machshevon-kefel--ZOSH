# מחשבון כפל גמלאות — סעיף 320

כלי לפקידים להשוואה בין גמלת שאירים לנכות כללית במצב של כפל זכאות.
חישוב מדויק לפי חוזר תשלומים 33, סכומים מעודכנים 2026.

## טכנולוגיה
React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui

## הרצה מקומית
```
npm install
npm run dev
```

## בנייה
```
npm run build
```

## מבנה
- `src/components/` — טפסים, טבלאות, Header
- `src/utils/calculations.ts` — לוגיקת חישוב
- `src/utils/constants.ts` — סכומים מעודכנים
