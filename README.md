# כפל גמלאות — בחירה בין נכות לשאירים

**קהל:** פקידים, אזרחים בעת שכול | **מצב:** אפיון

## מטרה
הגדרת תהליך מובנה לסיוע בבחירה בין גמלת נכות כללית לקצבת שאירים,
כאשר אזרח זכאי לשתיהן אך רשאי לקבל רק אחת.

## רקע
מקרה "כפל גמלאות" קורה לרוב לאחר פטירת בן/בת זוג.
הבחירה משפיעה על הכנסה לטווח ארוך ואינה הפיכה בקלות.

## מבנה התיקיות
```
כפל-גמלאות/
├── docs/          ← ניתוח משפטי, תרחישים, כלי בחירה
├── data/          ← נתוני השוואת סכומים
├── outputs/       ← פלטים מאושרים
└── experiments/   ← ניסויים
```

## הרצה
אין קוד — זהו פרויקט אפיון.
